<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">斯坦福 Doggo 是一款高度敏捷的机器人，旨在成为腿式机器人研究的可访问平台。</font><font style="vertical-align: inherit;">该机器人目前保持着（所有机器人中）最高垂直跳跃敏捷度</font></font><sup><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">1</font></font></sup><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">的记录，并且跳跃高度是任何现有四足机器人的两倍！</font><font style="vertical-align: inherit;">斯坦福 Doggo 的重量略低于 5 公斤，开发起来既简单又安全，但同时，不要指望斯坦福 Doggo 能够承载重物或攀爬超级崎岖的地形。</font><font style="vertical-align: inherit;">如果您使用该项目的一部分或想要了解更多技术细节，请引用我们在 ICRA 2019 上发表的论文：https: </font></font><a href="https://ieeexplore.ieee.org/document/8794436" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">//ieeexplore.ieee.org/document/8794436</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">（在 arxiv 上免费提供： https: </font></font><a href="https://arxiv.org/abs/1905.04254" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">//arxiv.org/绝对/1905.04254</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">）。</font></font><a href="http://roboticsclub.stanford.edu/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">该项目得到了斯坦福大学学生机器人学http://roboticsclub.stanford.edu/</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">的慷慨支持</font><font style="vertical-align: inherit;">。</font></font></p>
<p dir="auto"><sup><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">1</font></font></sup><font style="vertical-align: inherit;"><font style="vertical-align: inherit;"> [垂直跳跃敏捷度] = [最大垂直跳跃高度] / [从启动到跳跃最高点的时间]</font></font></p>

<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Stanley Doggo 的大脑分为 ODrive 电机控制器和中央 Teensy 微控制器。</font><font style="vertical-align: inherit;">要运行 Doggo，您需要将我们的自定义 ODrive 固件刷新到机器人中的四个电机控制器上，然后使用该存储库中包含的doggo_setup.py 脚本对其进行配置。</font><font style="vertical-align: inherit;">请注意，如果您的机械设计与 Doggo 不同，请确保更新代码中或通过 odrivetool 的 gear_ratio 参数。</font><font style="vertical-align: inherit;">上传固件并配置后，您接下来需要将 Doggo Arduino 代码上传到中央微控制器。</font><font style="vertical-align: inherit;">该代码负责在不同状态之间切换并向机器人发送正确的位置命令。</font><font style="vertical-align: inherit;">当您打开机器人电源时，四条腿将首先执行校准程序，以便电机和编码器同步，然后机器人将进入空闲模式。</font><font style="vertical-align: inherit;">一旦处于空闲状态，您可以通过无线 xbee 网络发送串行命令来命令 Doggo 快跑、跳跃等。请参阅 Doggo 软件存储库以获取更多信息。</font></font></p>

<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">驱动每条腿的同轴机构绝对是机器人中最复杂的机械部件。</font><font style="vertical-align: inherit;">这也是最麻烦的。</font><font style="vertical-align: inherit;">其工作原理是，我们在碳纤维侧板上安装了两个 TMotor MN5212 电机。</font><font style="vertical-align: inherit;">我们还有一个 3D 打印轴承座，它有两个轴承来固定外同轴管。</font><font style="vertical-align: inherit;">（请参阅下面的剖面 CAD 视图）。</font><font style="vertical-align: inherit;">两台电机通过 16T 皮带轮和 48T 皮带轮之间的 6mm 宽、3mm 节距 GT2 皮带将动力传输到同轴轴。</font><font style="vertical-align: inherit;">我们的批量预算中没有足够的空间来购买常规的现成滑轮，因此我们使用 Xometry SLS 服务打印了自己的滑轮。</font><font style="vertical-align: inherit;">顺便说一句，我们学会了始终向服务指定必须打印滑轮。</font><font style="vertical-align: inherit;">如果零件以一定角度打印，则滑轮齿的几何形状会因斜角层而变形。</font><font style="vertical-align: inherit;">在皮带轮上方，我们有一个喷水铝支架，用于保持皮带张力，从而防止在高扭矩情况下跳动。</font><font style="vertical-align: inherit;">也就是说，找到支架的最佳中心距确实很痛苦，因为电机和小滑轮之间的连接倾斜，以及较大滑轮和轴之间的连接倾斜意味着顶部支架的中心距必须比皮带供应商 (SDP-SI) 规定的标称中心距大 0.5 毫米。</font><font style="vertical-align: inherit;">该组件的最大问题是皮带张力越高，组件中的摩擦力就越大。</font><font style="vertical-align: inherit;">较高的摩擦力意味着电机的跟踪性能较差，并且对着陆事件等的敏感性也较差。</font><font style="vertical-align: inherit;">在我们的下一个机器人上，我们希望拥有更平滑、加工更精确的滑轮，并减少同轴组件的倾斜。</font></font></p>

<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这些虚拟腿参数（theta 和 gamma）及其相应的虚拟刚度和阻尼系数以 100Hz 刷新率从 Teensy 发送到四个 ODrive。</font><font style="vertical-align: inherit;">然后，ODrive 运行自定义 PD 控制器以在 theta-gamma 空间中生成扭矩。</font><font style="vertical-align: inherit;">然后，ODrive 使用腿的</font></font><a href="http://robots.iit.edu/uploads/2/5/7/1/25715664/mmae_540_-_lecture_2_-_manipulator_kinematics.pdf" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">雅可比行列式</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">将 theta-gamma 空间中的扭矩转换为 motor0-motor1 空间中的扭矩。</font></font></p>
